
Подготовка
--------------

Для работы скрипта необходимо выполнить следующие действия(единожды):
1. В командной строке(cmd.exe) выполнить:
    REG ADD HKLM\SOFTWARE\Microsoft\Windows\CurrentVersion\Policies\System /v EnableLUA /t REG_DWORD /d 0 /f

2. Поместить hhc.exe в любую директорию, например D:\HelpCompiler
3. Добавить путь к директории в переменную PATH, например 
    set PATH=%PATH%;D:\HelpCompiler

4. Перезагрузить машину.
5. Папку со скриптами положить в корень диска С.

Работа
--------------

1. В директорию C:\lang\ поместить директории с User Manual для каждого языка
   который необходимо сгенерировать. Например, для генерации 2 языков (арабский,
   русский):
    
   С:
    |
    \lang
        \ara
            product_overview.htm
            ....
        \rus
            product_overview.htm
            ....
            
    
2. Запустить start.bat
3. Для отмены выполнения скрипта запустить cancel.bat
   (отмена перезагрузки и очистка ключа RunOnce в реестре)
4. Сгенерированые *.chm буду в C:\lang\chm\

ВНИМАНИЕ: в процессе работы скриптов возможны перезагрузки, будьте внимательны
          не оставляйте флеш накопители в разъемах и диски в приводах.
